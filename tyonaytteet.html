RavintolaLaskuri

Raspberry Pi 5 -pohjainen kävijälaskentajärjestelmä, jota olen suunnitellut ravintolaympäristöön (soveltuu toki moneen muuhunkin ympäristöön), joka laskee reaaliaikaisesti sisään- ja uloskäynnit ravintolaympäristössä ilman internetyhteyttä. Projektiin kuuluu Flask-palvelin, liikkeentunnistukseen perustuva laskentaohjelma ja React Native -mobiilisovellus.

Projekti on vielä hieman kesken, Flask-palvelin toimii jo moitteettomasti, palvelin käynnistyy heti kun raspberry kytketään päälle ja mobiilisovellus kommunikoi moitteetta python pohjaisen laskennan kanssa. Itse laskenta vaatii vielä hienosäätöä asetuksilla.

Nettikauppa

Toteutin verkkokauppajärjestelmän PHP:llä ja MySQL-tietokannalla. Sovellus sisältää käyttäjähallinnan, ostoskorin, kassatoiminnon sekä erillisen admin-näkymän tilausten ja tuotteiden hallintaan. Järjestelmässä on käyttöoikeusmalli (käyttäjä / admin), AJAX-pohjainen ostoskori ilman sivun uudelleenlatauksia sekä tilausprosessi, joka tallennetaan tietokantaan. Admin-näkymässä tilauksia voidaan tarkastella ja niiden tilaa (uusi, käsittelyssä, toimitettu, peruttu) hallita. Projektista on tehty valmis nettikaupan pohja ilman oikeita tuotteita, sekä tietoturvaan on kiinnitetty huomiota (prepared statements, session-hallinta, roolit). Teknologiat: PHP, MySQL (PDO), JavaScript (Fetch API / AJAX), HTML, CSS
